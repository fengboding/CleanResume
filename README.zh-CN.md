# CleanResume

CleanResume 是一款基于原生 JS 构建的响应式简历生成器，主打隐私安全与实时预览。在浏览器内即可完成排版、定制模板并导出高清 PDF，所有数据本地处理，绝不上传。

---

## 核心特性

**排版与定制**
- **实时 A4 预览**：在自适应缩放的 A4 画布上即时预览排版效果。
- **20 款布局模板**：提供20种个人资料对齐、分割线位置、图标可见性及配色规则的组合方案。
- **排版微调**：通过滑块精细调整字号、行高、页面边距与段落间距。
- **自定义水印**：支持添加文本水印，可调节透明度、字号及 X/Y 偏移量。
- **明暗主题**：应用界面支持 shadcn/ui 风格的明暗模式切换（预览纸张始终保持白底打印样式）。

**编辑与交互**
- **拖拽排序**：通过鼠标或触摸手势，轻松对段落、条目和标签进行拖拽排序。
- **富文本编辑**：在描述字段中使用类 Markdown 语法输入无序列表（`- 内容`）和加粗文本（`**文本**`）。
- **定制化输入**：专用的时间段双输入框和标签输入框（按 `Enter` 添加标签）。
- **显隐控制**：独立控制特定段落是否在预览中显示，无需删除数据。

**数据与导出**
- **一键导出 PDF**：导出高保真 A4 尺寸 PDF，文件名自动包含姓名、职位与日期，支持开启“单页”约束。
- **JSON 支持**：无缝导入与导出 `.json` 格式的简历数据。
- **拖拽导入**：将 `.json` 文件拖入页面即可导入，弹窗会详细展示字段匹配、部分匹配及忽略的校验结果。
- **自动保存**：所有简历数据、界面状态与设置均本地存储，自动跨会话保留。

## 语言特性 (`zh-CN`)

CleanResume 启动时将自动检测系统语言并适配界面。选择简体中文时：

- **默认段落**：教育背景、工作经历、项目经验、荣誉奖项、资格证书、专业技能、语言能力。
- **示例数据**：加载包含本土姓名、公司与学校名称的完整中文简历示例。

## 使用指南

### 1. 内容编辑
- **打开编辑器**：点击右侧的“编辑”浮动按钮滑出编辑面板。
- **添加内容**：点击底部的“添加”按钮创建新段落，在段落内点击“+”图标添加条目。
- **格式化文本**：在描述字段中输入 `- ` 创建要点，输入 `**文本**` 加粗文字。

### 2. 样式设置
- **打开设置**：点击“设置”浮动按钮滑出设置面板。
- **选择模板**：点选缩略图卡片切换基础布局。
- **应用配色**：选取主题色，根据模板规则将其应用到姓名、标题、图标或标签。
- **调整版式**：通过滑块调整排版、间距、边距及水印配置。

### 3. 数据管理
- **导入**：点击“导入”或直接将 `.json` 文件拖入页面，通过校验表格确认数据匹配情况。
- **导出 JSON**：点击“导出”下载数据的 `.json` 备份文件。
- **保存 PDF**：点击“下载”按钮生成 PDF。若开启“单页”选项，内容将自动缩放以适应单页。

## 使用技巧

- **折叠面板**：点击段落或条目旁的折叠图标将其收起，保持编辑界面整洁。
- **智能主键**：新建条目默认显示随机 ID，输入学校或公司名称后，ID 会自动替换为该名称。
- **导入校验**：在导入校验弹窗中点击任意行，可展开查看具体匹配的字段详情。
- **空状态提示**：简历为空时，预览区会循环播放设置引导提示。
